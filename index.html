The snowcovered areas have been classified from images captured by the MODIS instrument on board the Terra (EOS AM) satellite. A method called the Normalised Difference Snow Index (NDSI) was used to delineate snow covered areas which were then clipped to the Snowy Water Catchment.

The NDSI algorithm and subsequent geoprocessing was automated in a QGIS processing toolbox model - a case study regarding this can be found here and the QGIS processing model can be found on GitHUB.
